Внимание, это задание НУЖНО решать с использованием обычных группировок и агрегатных функций

Вам нужно найти размер и медиану некоторой выборки численных значений.


Подробности

Медиана упорядоченной выборки с нечётным количеством элементов -- это элемент, стоящий ровно посередине выборки.
Например, для выборки [1, 1, 2, 3, 100, 100500, 1233456789] медианой будет число 3.
Вам нужно найти медиану и размер (общее количество элементов) выборки, записанной в таблице.

Это задание подразумевает использование группировок и агрегатных функций, что неудивительно, учитывая название модуля.
Его в принципе можно решить разными способами, не использующими группировки.
Постарайтесь их не использовать. Довольно простой запрос с не бог весть какими группировками прекрасно всё сделает.


Выборка хранится в таблице T(id INT PRIMARY KEY, value INT UNIQUE).
Таблица заполняется нечетным количеством случайных чисел. Дубликатов в выборке нет.

Ответ и проверка

Ответом на задание будет SQL-запрос, возвращающий одну строку с двумя столбцами.
В первом столбце должно быть записано значение медианы, во втором размер выборки, оба значения должны быть целочисленными
